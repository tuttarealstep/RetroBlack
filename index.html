  • The grid system is like the bootstrap system. It have rows and columns, rows are used for create horizontal groups of columns.
  • Rows must be placed within a .container (fixed-width) or .container-fluid (full-width) or .container-full (full-width, full-height) for proper alignment and padding.

                
<div class="container">
    <!-- For each row, give the class "row" -->
    <div class="row">
         <!-- for columns, add the class column-x, replace x with a number from 1 to 12 -->
            <div class="column-6">six</div>
            <div class="column-6">six</div>
    </div>
</div>

BUTTONS

RetroBlack come with a large number of buttons with many styles and classes.

You can use button classes on <button>,<input> and <a>.

Link 
                    
<input type="submit" class="button button-default" value="Submit">
<input type="button" class="button button-default" value="Input">
<a class="button button-default" href="#" role="button">Link</a>
<button class="button button-default" type="submit">Button</button>

Others styles

                
<!-- only button class -->
<input type="submit" class="button" value="only button class">

<!-- normal button -->
<input type="submit" class="button button-normal" value="normal">

<!-- normal button -->
<input type="submit" class="button button-default" value="default">

<!-- red button -->
<input type="submit" class="button button-red" value="red">

<!-- green button -->
<input type="submit" class="button button-green" value="green">

<!-- blue button -->
<input type="submit" class="button button-blue" value="blue">

<!-- orange button -->
<input type="submit" class="button button-orange" value="orange">

<!-- clear button -->
<input type="submit" class="button button-clear" value="clear">

Sizes

You want smaller or larger buttons? Its simply add these classes:
.button-extra-small, .button-small, .button-large or .button-extra-large

                    
<p>
   <input type="submit" class="button button-normal button-extra-small" value="extra small">
</p>

<p>
   <input type="submit" class="button button-default button-small" value="small">
</p>

<p>
  <input type="submit" class="button button-red " value="default">
</p>

<p>
  <input type="submit" class="button button-green button-large" value="large">
</p>

<p>
  <input type="submit" class="button button-blue button-extra-large" value="extra large">
</p>

If you like a large button with the full width of its parent add this class:.button-block


                    
<input type="submit" class="button button-normal button-block" value="button-block">

<input type="submit" class="button button-default  button-block" value="button-block">

You can also add an .active and .disabled state to the button.


                
<input type="submit" class="button button-normal active" value="active">

<input type="submit" class="button button-default active" value="active">

<input type="submit" class="button button-normal disabled" value="disabled">

<input type="submit" class="button button-default disabled" value="disabled">

Forms

Forms are pretty difficult to organize and style. But RetroBlack can help you with its classes.


You can define some .form-block a group of label and input. They help you with the space.

For every text input or selection, if you add the .form-element class it become with a beautiful style.

Help text.


                    
<form>
    <div class="form-block">
        <label for="email">Email</label>
        <input type="email" class="form-element" id="email" placeholder="Email">
    </div>

    <div class="form-block">
        <label for="password">Password</label>
        <input type="password" class="form-element" id="password" placeholder="Password">
    </div>

    <div class="form-block">
        <label for="file">Load File</label>
        <input type="file" id="file">
        <p class="help-text">Help text.</p>
    </div>

    <div class="checkbox">
        <label>
            <input type="checkbox"> Check box
        </label>
    </div>

    <div class="form-block">
        <label for="state">State</label>
        <select id="state" class="form-element">
            <option>AL</option>
            <option>CA</option>
            <option>IL</option>
        </select>
    </div>

    <button type="submit" class="button button-normal">Send</button>
    <button type="reset" class="button white-button-red">Reset</button>
</form>


But if you like a minimal style for your .form-element,
simply replace:.form-element class with this:.minimal-form-element.

Utilities

RetroBlack have some utilities class, for help you with the text, align...


                    
<!-- Pulls element to the right -->
<div class="float_right"></div>

<!-- Pulls element to the left -->
<div class="float_left"></div>

<!-- Makes the text aligned center -->
<div class="text-center"></div>

<!-- Makes the text aligned left -->
<div class="text-left"></div>

<!-- Makes the text aligned right -->
<div class="text-right"></div>

<!-- justify the text -->
<div class="text-justify"></div>

<!-- capitalize the text -->
<div class="text-capitalize"></div>

<!-- make the text uppercase -->
<div class="text-uppercase"></div>

<!-- no wrap the text -->
<div class="text-nowrap"></div>

<!-- make the text small -->
<div class="text-small"></div>
